# StudyShare 🎓

StudyShare – A creative learning and sharing space for GSFC University students.

## Deployment
- **Live Site:** [https://study-share.vercel.app](https://study-share.vercel.app)
- **Old Link:** [https://tinkering-mentoring.vercel.app](https://tinkering-mentoring.vercel.app) (Redirects to new link)

## Local Development
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env`.
4. Run locally: `npm run dev`
